FAQ

Is this on the Chrome Web Store?

Not yet — for now, install it as an unpacked extension. Setup instructions are in the GitHub README.

Do I need my own AI API key?

No — tailoring calls run through a server-side Edge Function using the operator's key. You just pick a provider and model in Settings.

Is my resume data private?

Your resume and job data are stored in Postgres with row-level security scoped to your account — no other user can query it. See the Privacy Policy for details.

Is this open source?

Yes, MIT-licensed open source — the full source, including the database schema and Edge Function, is on GitHub.
